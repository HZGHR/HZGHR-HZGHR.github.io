工作经历

（上汽）环球车享成都汽车租赁有限公司

人事专员 　　　　　　　　　　2018.06 - 2020.02

一、招聘工作

  1. 发布招聘信息、简历筛选、候选人面试、签订劳动合同；
  2. 开拓和维护付费、免费、政企、校企等招聘渠道；
  3. 组织、实施招聘会：校园专场/双选招聘会及社会招聘会；
  4. 负责试用期人员考核及转正；
  5. 针对各岗位进行访谈，编撰岗位说明书。

二、绩效工作

  1. 月度绩效、季度绩效、年度绩效各部门打分及汇总；
  2. 根据业务需求及员工工作内容，修订绩效评分指标体系；
  3. 对低绩效员工进行绩效谈话，寻找问题，提供帮助。

三、企业文化建设

  1. 搭建企业内部文宣平台，创建并运营内部微信公众号，设计维护钉钉企业平台；
  2. 强化企业内控管理，个人自购服务器及域名，创建公司制度学习及查询平台；
  3. 策划组织团建、拓展、年会、生日会、足球赛等特色主题活动，为员工减压，增强团队凝聚力；
  4. 撰写新闻稿、通知启事、会议纪要、活动策划等文案；
  5. 开展员工关怀，组织开展公司月度/季度员工生日会，制作员工生日钉钉开屏页，协助党支部慰问困难职工等。

四、培训与开发

  1. 组织内部员工担任讲师，进行常态化内训；
  2. 担任讲师，讲授新员工培训、内控管理等课程；
  3. 外部培训对接落地及流程把控；
  4. 培训效果考核及反馈收集。

五、员工关系

  1. 员工入、转、调、离相关手续及流程办理；
  2. 员工信息登记、保管及更新；
  3. 建立内部沟通渠道，及时收集和反馈员工意见建议；
  4. 为员工提供有关国家法律、法规、公司政策及管理制度等方面的咨询服务。

六、其他

  1. 起草、修订制度及管理办法；
  2. 制作公司汇报PPT、活动PPT，剪辑活动视频、音频；
  3. 公司打印机、电脑等办公设备维护、网络维护；
  4. 以低成本搭建公司视频/电话会议硬件系统，提升远程会议体验感和便捷性。

4、对接人员外包项目，处理外包人员手续、入职及住宿；
5、招聘数据汇总分析，每日汇报。

4、劳动合同续订管理；

通威太阳能有限公司

人事专员　　　2020.12 - 至今

在职期间主要负责人才招募、人才发展、员工关系、数据支持、内部培训。

一、人才招募

1、负责技术类、管理类、职能类岗位招聘，累计招聘推荐260余人；
2、编制（HC）管理，调整各部门编制，协助业务部门组织架构调整转型；
3、招聘渠道开发及维护，协商招聘需求，提前做好业务人才储备。

二、人才发展

1、定期策划组织年中/年终晋升，单次参与人数150-230人，负责全程管理（资格初审、面试/笔试、360测评、名单确认、系统调动）；
2、关联培训项目，不定期组织储备技术员/储备组长晋升，促进人才发展；
3、根据业务发展进度，与业务部门协商，不定期举行管理人才晋升；
4、向培训提供人才培养和人才储备建议，协调培训开设储备培养班。

三、员工关系

1、每周离职面谈，了解员工离职原因，处理员工投诉及业务建议；
2、处理劝退、辞退、试用期不合格淘汰员工，降低法律风险；
3、每周业务下沉，了解部门内部情况，向部门提供管理改善建议。

四、数据支持

1、ERP系统人事流程维护，对接IT完善功能，开发新流程等；
2、离职数据分析，分析离职原因、离职趋势、群体特征等字段，提供离职预警及改善建议；
3、每周缺编统计，追踪最新人员缺口及分布，动态调整招聘计划；
4、定期完成周报/月报，向数据组提供招聘数据；
5、撰写对公人力资源信息文件，制作部门领导PPT，设计制作海报/H5/推文等各类资料。

五、内部培训

1、独立开发四地招聘组课程《快速发展期的人才引进及配置》，并组建四地专题讲师队伍；
2、组内新人带教，带领新人熟悉业务，解答新人人力资源相关问题；
3、人力资源部内部培训《五型人格（PDP心理测试）》、《招聘渠道特点及维护》。
